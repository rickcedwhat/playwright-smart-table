Failure states ​

const row = table.getRow({ Name: 'Nobody' });
await expect(row).not.toBeVisible();

Sentinel row: valid Locator, no DOM match.

row.getCell('DoesNotExist')

Throws when the column is not in the header map.

await table.findRow({ ... }, { maxPages: 1 })

Returns sentinel when no match within maxPages.
