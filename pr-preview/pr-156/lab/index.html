Inline components ​

MethodBadge ​

Inline async/sync pill for use next to method names in API docs.

getRow() sync returns the first matching row from the current page without paginating.

findRow() async paginates until a match is found or maxPages is exhausted.

findRows() async collects all matching rows across pages.

init() async resolves headers and populates the column map.

ConfigSwatch ​

Collapsible config block for embedding ready-to-use configs inline in example pages.
