# Query Your Table

Once Playwright Smart Table understands your table, here's what you can ask it to do.

- [Find rows](/guide/query/find-rows) — `getRow`, `findRow`, `findRows`
- [Read cells](/guide/query/read-cells) — `getCell`, `toJSON`
- [Iterate rows](/guide/query/iterate) — `forEach`, `map`, `filter`
- [Write to cells](/guide/query/write) — `smartFill`
