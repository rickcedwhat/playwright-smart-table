# API Reference

Complete reference for all `useTable` configuration options, table methods, SmartRow methods, and pluggable strategies. Signatures are auto-injected from source on each build.

If you're new, start with the [Quick Start guide](/guide/start) and come back here when you need the full signature for a specific method.

---

## Reference pages

| Page | What it covers |
|---|---|
| [Config Options](/api/table-config) | All `useTable(locator, config)` options — selectors, behavior, hooks, strategies |
| [Table Methods](/api/table-methods) | Methods on the initialized table — `findRow`, `forEach`, `map`, `sorting`, etc. |
| [SmartRow](/api/smart-row) | Methods on a row object — `getCell`, `toJSON`, `smartFill`, `bringIntoView` |
| [Strategies](/api/strategies) | All pluggable strategies — pagination, loading, viewport, fill, dedupe, and more |
